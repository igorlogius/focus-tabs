Overview

This addon lets you setup instant focus mappings (brings tabs into focus)
All you need is to assign a tab to a focus shortcut and as long as the
tab has not been closed or the assigment changed, the tab will be brought
into focus via the corresponding focus shortcut

Setup
  1. Set assigment and focus shortcut keys here:
    1. Open about:addons (Ctrl+Shift+A)
    2. Click Gear Icon (Top Right)
    3. Click Manage Extension Shortcuts (End of List)
  2. Set the Assign Keys to to use for mapping the focus keys to tabs
  3. Set the Focus Keys to switch to the previously assigned tab
Note: In the future this popup might show which shortcuts
are assigned to still existing tabs (id,index,url)
